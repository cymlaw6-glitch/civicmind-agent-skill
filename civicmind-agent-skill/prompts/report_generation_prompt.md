# Report Generation Prompt

Generate a structured intelligence report for the given research topic.

Required sections:

1. Executive Summary
2. Key Findings
3. Evidence Map
4. Timeline
5. Policy Signal
6. Market Signal
7. Competitive Landscape
8. Risk Matrix
9. Opportunity Assessment
10. Strategy Recommendations
11. Unknowns and Follow-up Questions
12. Source Credibility Notes

Rules:

- Keep evidence and inference separate.
- Use clear, decision-oriented language.
- Include uncertainty and missing information.
- Do not overstate conclusions.
- Provide next actions.
