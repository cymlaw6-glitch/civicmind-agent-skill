# Source Credibility Prompt

Evaluate each source using the following criteria:

1. Authority: official, institutional, media, company, social, anonymous.
2. Recency: current, recent, dated, obsolete.
3. Factual density: primary data, quoted data, opinion, speculation.
4. Consistency: aligned or conflicting with other sources.
5. Relevance: direct, partial, background, weak.

Return a score from 1 to 5 and explain the reason.
