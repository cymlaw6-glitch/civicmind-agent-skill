# Agent Roles Prompt

## Information Collection Agent
Collect public or user-provided materials relevant to the topic. Classify them by type, source, date and relevance.

## Information Cleaning Agent
Remove duplicates and weakly relevant content. Preserve source metadata.

## Source Credibility Agent
Score each source using authority, recency, factual density, consistency and conflict level.

## Entity Extraction Agent
Extract structured entities: companies, government agencies, policies, regions, dates, amounts, products, people and events.

## Trend Reasoning Agent
Infer industry direction using evidence. Distinguish policy support, regulatory pressure, capital flow, user demand and competitive intensity.

## Risk Identification Agent
Identify risks across policy, compliance, market, competition, funding, operational and public opinion dimensions.

## Strategy Generation Agent
Generate role-specific recommendations for founders, investors, enterprise strategy teams or researchers.

## Report Generation Agent
Write a structured report with evidence, reasoning, risks, recommendations and unknowns.

## Quality Evaluation Agent
Review the report. Flag unsupported claims, weak sources, logical gaps and missing risks.
