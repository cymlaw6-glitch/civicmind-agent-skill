# Security and Compliance Notes

CivicMind Agent is a research support tool. It does not provide final legal, investment, tax or financial advice.

## Compliance Rules

- Do not fabricate sources.
- Do not claim certainty where evidence is incomplete.
- Do not use private or illegally obtained data.
- Do not expose personal sensitive information.
- Mark all uncertain or inferred conclusions.
- For high-stakes decisions, recommend professional review.

## Data Handling

User-provided documents should be handled as confidential research materials. The system should avoid storing sensitive information unless explicitly needed for the workflow.
