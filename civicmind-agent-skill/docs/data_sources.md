# Data Sources

CivicMind Agent is designed to work with public and user-provided information.

## Source Categories

| Category | Examples | Credibility Weight |
|---|---|---|
| Government sources | official notices, policy documents, regulatory announcements | Very high |
| Exchange filings | annual reports, prospectuses, exchange announcements | Very high |
| Company sources | official websites, product pages, press releases | Medium to high |
| Research reports | broker research, consulting reports, industry white papers | Medium to high |
| News media | mainstream media, industry media | Medium |
| Social media | posts, forums, comments | Low, sentiment only |
| Expert commentary | interviews, conference notes, podcasts | Context-dependent |

## Source Handling Principles

- Prefer primary sources.
- Record publication dates.
- Separate facts from opinions.
- Mark conflicts between sources.
- Avoid relying on one source for major conclusions.
- Use social media only as a weak signal unless verified.
