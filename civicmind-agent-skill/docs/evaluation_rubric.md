# Evaluation Rubric

CivicMind Agent evaluates each report across six dimensions.

| Dimension | Description | Score |
|---|---|---|
| Evidence completeness | Are key claims supported by sources? | 1-5 |
| Source quality | Are high-quality sources prioritized? | 1-5 |
| Logical consistency | Do conclusions follow from facts? | 1-5 |
| Risk coverage | Are major risks and unknowns included? | 1-5 |
| Role fit | Does the report match the user's decision role? | 1-5 |
| Actionability | Are recommendations specific and practical? | 1-5 |

## Failure Conditions

A report should be flagged if it:

- gives definitive conclusions without evidence;
- hides uncertainty;
- uses outdated sources for current decisions;
- confuses opinion with fact;
- gives investment/legal/tax advice without caveats;
- fails to include risk analysis;
- fails to identify missing information.
